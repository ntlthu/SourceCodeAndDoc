- Live SASS
https://marketplace.visualstudio.com/items?itemName=ritwickdey.live-sass

1. Cài đặt đường dẫn tạo file css bằng visual code:
 Extensions → Live Sass Compiler → Extension Settings → Edit in settings.json

2. Dán đoạn code bên dưới vào cuối file setting
"liveSassCompile.settings.formats":[
        // This is Default.
        {
            "format": "expanded",
            "extensionName": ".css",
            "savePath": "~/../css/"
        }        
    ]

3. Koala (nếu không dùng được live sass) :
